# Cogs vs Clips Starter Policy

This starter is an editable Cogs vs Clips policy generated by Coworld.

Build it with:

```bash
docker build --platform=linux/amd64 -t cogs_vs_clips:latest .
```

Then test it against a downloaded Cogs vs Clips Coworld. After `uv run coworld download cogs_vs_clips`, replace
`<coworld-id>` with the printed `Coworld:` ID:

```bash
uv run coworld run-episode ./coworld/<coworld-id>/coworld_manifest.json cogs_vs_clips:latest --timeout-seconds 120
```

The policy source is `player.py`. It connects to `COWORLD_PLAYER_WS_URL`, validates `player_config` and `observation`
messages, and returns legal movement or noop actions.

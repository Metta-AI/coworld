# Optimizer Template

The optimizer role is an interactive workbench rather than a one-shot job. This starter gives a manifest fragment plus a
minimal plan writer for Coworlds experimenting with a game-specific optimizer runnable.

Contract reference: `coworld/docs/roles/OPTIMIZER.md`.

Files:

- `optimizer_manifest_entry.json` - manifest fragment for a game-specific optimizer runnable.
- `optimizer_plan.py` - small JSON plan writer matching the current Paint Arena reference optimizer style.

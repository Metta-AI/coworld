# Evidence-First Debugging

1. Collect evidence before editing.
2. Name the failing capability.
3. Find the smallest owner.
4. Patch one behavior.
5. Add a focused regression.
6. Rerun the same evidence path.
7. Record the result, including negative or incomplete outcomes.

# CrewRift Social Diagnosis

1. Fetch the full completed artifact set for the request or round.
2. Identify role, score, task progress, deaths, reports, votes, and policy logs.
3. Classify one low-row failure in behavioral terms.
4. Check CrewRift source for the relevant mechanic.
5. Route the failure to its source owner.
6. Patch one behavior and add a narrow regression.
7. Re-run the same evidence path.
8. Record the decision in the candidate ledger or performance log.

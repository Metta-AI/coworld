# Remote XP Loop

1. `uv run --no-sync --project . co-gas mandate refresh-report`
2. `uv run --no-sync --project . co-gas mandate check`
3. Choose the failing league and lower owned lane.
4. Inspect completed artifacts or run focused diagnostics until one failure is
   named.
5. Patch source under a source-custody path.
6. Run focused checks and update candidate YAML.
7. Upload no-submit with `co-gas submit-source --no-submit`.
8. Request XP against public leaders, lower owned champion, and recent bad
   matchups with seat rotation where available.
9. Poll to terminal and fetch full artifacts.
10. Promote the exact supported policy version, or record a hold.

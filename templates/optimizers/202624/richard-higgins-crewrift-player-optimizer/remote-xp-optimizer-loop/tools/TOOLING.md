# co-gas Tooling

- Live state: `co-gas mandate refresh-report`, `co-gas mandate check`.
- Source custody: `co-gas sources validate`.
- Candidate records: `co-gas candidates validate`, `experiments/candidates/*.yaml`.
- Local diagnostics: CrewRift slot matrix runner, case event summaries, policy
  tests, and source assertions.
- Hosted policy versions: `co-gas submit-source --no-submit`.
- XP: `co-gas xp create`, `co-gas xp status`.
- Promotion: submit the exact policy version selected by completed XP.

# Cogs vs Clips Coworld

This example packages the Cogsguard Cogs vs Clips game as a Coworld-compliant Cogame.

## Build Images

From the repo root:

```bash
docker build -f packages/coworld/examples/cogs_vs_clips/game/Dockerfile -t coworld-cogs-vs-clips-game:latest .
docker build -t coworld-cogs-vs-clips-noop-player:latest packages/coworld/examples/cogs_vs_clips/player
```

## Play Locally

From the Coworld package root (`packages/coworld`):

```bash
uv run coworld play examples/cogs_vs_clips/coworld_manifest.json
```

Open the printed `/global` URL to watch the live MettaGrid replay stream. The game image serves a websocket-capable
MettaScope build that is generated by the Dockerfile; the page also includes a simple canvas fallback. Open any printed
`/player?slot=...&token=...` URL to play that cog from the browser. Slots that are not connected advance with `noop`.

## Certify

From the Coworld package root (`packages/coworld`):

```bash
uv run coworld certify examples/cogs_vs_clips/coworld_manifest.json
```

## View A Replay

After `play` or `certify` writes a replay artifact, start a replay viewer from the Coworld package root
(`packages/coworld`):

```bash
uv run coworld replay examples/cogs_vs_clips/coworld_manifest.json path/to/replay.json
```

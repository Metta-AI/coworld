# Coworld

Coworld is the public CLI and Python package for Softmax v2 tournaments. A Coworld packages a game container, policy
containers, runtime docs, schemas, configs, and replay/result contracts into one runnable tournament unit.

The package documentation lives in:

- [COWORLD_README.md](src/coworld/COWORLD_README.md): concepts and end-to-end workflows
- [CLI_README.md](src/coworld/CLI_README.md): command reference
- [GAME_RUNTIME_README.md](src/coworld/GAME_RUNTIME_README.md): game-container runtime contract

Install in a project with:

```bash
uv add "coworld[auth]"
```
